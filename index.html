How can you make something to animate?

  1. Get an SVG
  2. Has to have a stroke, dash offset, maybe dasharray
  3. Animate dash offset with keyframes
					
						
						.path {
							stroke-dasharray: 1000;
							stroke-dashoffset: 1000;
							animation: dash 5s linear forwards;
						}
						
						@keyframes dash {
							to {
								stroke-dashoffset: 0;
							}
						}
